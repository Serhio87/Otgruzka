Приложение позволяет пользователю добавлять продукцию, оформлять продажу продукции, дополнять необходимыми данными имеющиеся справочники, в будущем можно будет дополнить необходимым функционалом. 
Основные функции:
–	количественный и номенклатурный учет продукции;
–	просмотр отгруженной продукции;
–	добавление необходимых данных в справочники при поступлении новой продукции, нового покупателя или сотрудника;
–	продажа продукции со склада;
–	использование базы данных для хранения данных об операциях и нормативно-справочной информации в базе данных Microsoft Access;
–	обеспечение связи между операциями, остатками продукции и нормативно-справочной информацией через соответствующие таблицы базы данных.
